## 接口地址

| 操作 | URL |
|------|-----|
| 提交任务 | `https://visual.volcengineapi.com?Action=CVSync2AsyncSubmitTask&Version=2022-08-31` |
| 查询结果 | `https://visual.volcengineapi.com?Action=CVSync2AsyncGetResult&Version=2022-08-31` |

## 固定 Header 参数

| 参数 | 值 |
|------|-----|
| Service | `cv` |
| Region | `cn-north-1` |
| Content-Type | `application/json` |

鉴权方式：HMAC-SHA256 Signature V4

## req_key（模型标识）

| 版本 | req_key（用户控制台） | req_key（官方文档） | 说明 |
|------|----------------------|---------------------|------|
| 720P | `high_aes_general_v30l_zt2i` | `jimeng_t2v_v30` | 即梦3.0 文生视频 |
| 1080P | `high_aes_general_v30_zt2i` | `jimeng_t2v_v30_1080p` | 即梦3.0 1080P 文生视频 |
| Pro | `high_aes_general_v30_pro_zt2i` | `jimeng_ti2v_v30_pro` | 即梦3.0 Pro 文生+图生视频一体 |

## 时长参数（重要）

**`high_aes_general_v30*_zt2i` 系列 key 使用 `duration` 参数：**

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| `duration` | int | ❌ | 视频秒数，范围 **3~54**，默认 10。推荐 15 或 30 |

**官方 `jimeng_t2v_v30*` 系列 key 使用 `frames` 参数：**

| frames | 对应时长 |
|--------|---------|
| 121 | 5秒（24×5+1） |
| 241 | 10秒（24×10+1） |

> ⚠️ 两套 key 参数不同！`high_aes_general` 用 `duration`/`width`/`height`，`jimeng_t2v` 用 `frames`/`aspect_ratio`。

## 分辨率参数

`high_aes_general_v30*_zt2i` 系列使用 `width` / `height` 而非 `aspect_ratio`：

| 比例 | 720P | 1080P / Pro |
|------|------|-------------|
| 16:9 | 1280×720 | 1920×1080 |
| 9:16 | 720×1280 | 1080×1920 |
| 4:3 | 960×720 | 1440×1080 |
| 3:4 | 720×960 | 1080×1440 |
| 1:1 | 720×720 | 1080×1080 |
| 21:9 | 1680×720 | 2520×1080 |

## 三版差异对比

| 功能 | 720P | 1080P | Pro |
|------|------|-------|-----|
| req_key（控制台） | `high_aes_general_v30l_zt2i` | `high_aes_general_v30_zt2i` | `high_aes_general_v30_pro_zt2i` |
| 文生视频 | ✅ | ✅ | ✅ |
| 图生视频-首帧 | 需公网URL | 需公网URL | 支持公网URL+base64本地文件 |
| 图生视频-首尾帧 | 单独接口 | 单独接口 | 单独接口 |
| 图生视频-运镜 | 单独接口 | 单独接口 | 单独接口 |
| 分辨率 | 720P | 1080P | 1080P 专业级 |
| duration | 3~54秒 | 3~54秒 | 3~54秒 |
| 并发 | 2路 | 2路 | 2路 |

## 提交任务（Body 参数）— high_aes_general 系列

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| `req_key` | string | ✅ | 模型标识 |
| `prompt` | string | ✅ | 提示词，建议 ≤400字，不超过800字 |
| `duration` | int | ❌ | 3~54秒，默认 10 |
| `width` | int | ❌ | 视频宽度，默认 1280 |
| `height` | int | ❌ | 视频高度，默认 720 |
| `seed` | int | ❌ | 随机种子，默认 -1 |
| `image_urls` | array | 图生视频 | 图片公网 URL 数组 |
| `binary_data_base64` | array | 图生视频Pro | 图片 base64（Pro 版专属） |

### 文生视频请求示例（15秒）

```json
{
    "req_key": "high_aes_general_v30l_zt2i",
    "prompt": "Cinematic shot, Paris dusk...",
    "duration": 15,
    "width": 720,
    "height": 1280,
    "seed": -1
}
```

### 文生视频请求示例（30秒）

```json
{
    "req_key": "high_aes_general_v30l_zt2i",
    "prompt": "Cinematic shot, Paris dusk...",
    "duration": 30,
    "width": 720,
    "height": 1280,
    "seed": -1
}
```

### 图生视频请求示例（Pro）

```json
{
    "req_key": "high_aes_general_v30_pro_zt2i",
    "image_urls": ["https://xxx"],
    "prompt": "Dynamic product showcase...",
    "duration": 15,
    "width": 1080,
    "height": 1920,
    "seed": -1
}
```

### 返回示例（成功）

```json
{
    "code": 10000,
    "data": {
        "task_id": "7392616336519610409"
    },
    "message": "Success"
}
```

## 查询结果（Body 参数）

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| `req_key` | string | ✅ | 同提交时一致 |
| `task_id` | string | ✅ | 提交返回的 task_id |
| `req_json` | string | ❌ | JSON 序列化的隐式水印配置 |

### 返回示例（生成完成）

```json
{
    "code": 10000,
    "data": {
        "status": "done",
        "video_url": "https://xxx"
    },
    "message": "Success"
}
```

### 状态说明

| status | 说明 |
|--------|------|
| `in_queue` | 排队中 |
| `generating` | 生成中 |
| `done` | 完成 |
| `not_found` | 任务不存在或已过期 |
| `expired` | 已过期 |

## 重要规则

1. **优先判断 `code == 10000`**，再判 `data.status`
2. `video_url` **有效期为 1 小时**
3. Pro 版图生视频：图片 max 4.7MB, max 4096×4096, 仅 JPEG/PNG
4. 无免费额度，按量计费
5. **30秒以上视频容易人物畸变**，建议拆两段 15 秒生成后拼接，稳定性更高
6. 时长越长扣费越多，30秒约 3 倍于 10 秒费用
