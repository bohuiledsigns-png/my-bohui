AI Short Video Desktop Tool

通义万相 + TTS + BGM — AI 一键生成带货视频

📖 快速上手

选择产品图 → 选模板 → 调整参数 → 点击「✨ AI 自动」→「▶ 生成视频」

🎬 两种模式

模式操作适用场景
通义万相生成 ① 点产品图 → ③ 调参数 → AI 自动 → 生成视频 无素材时 AI 从零生成
导入已有视频 上传视频 → 填写产品名 → AI 配乐+字幕 已有拍摄素材，加配音/BGM/字幕

🎤 自定义声音

TTS 音色下拉选「🎤 自定义声音」→ 上传你的录音文件（.wav / .mp3 / .m4a）→ 从下拉列表选已上传的声音 → 生成视频时会直接用你的录音。

注意：这是直接使用你上传的音频文件，不是 AI 声音克隆。

🔊 保留原声

导入已有视频时，勾选「保留原声」则不替换音频轨道，仅叠加字幕和封面钩子文字。适合原视频已有讲解或背景声的场景。

⏱ 视频时长

可选 5秒 / 10秒 / 15秒。AI 会根据所选时长自动生成匹配长度的文案，字幕也按时长自动分段。

📄 参考链接仿写

粘贴 TikTok / 抖音 / B站 / YouTube / 小红书等链接 → 点「🔍 分析仿写」→ AI 自动分析风格并生成类似的 prompt / 旁白 / 钩子。

⚙️ 参数说明

参数说明
画质720P 默认 / 1080P 高清 / Pro 最高（生成更慢）
比例9:16 竖屏（抖音/TikTok）/ 16:9 横屏（YouTube）/ 1:1 方形
BGM轻奢钢琴 / 环境氛围 / 潮流电子 / 无 BGM
随机种子-1=随机，固定数字可复现相同画面
钩子标题视频开头 3 秒显示的封面文字，吸引点击

🐌 生成慢怎么办？

最耗时的是 通义万相视频生成（2-10 分钟），这是云端计算时间，无法跳过。如果已有视频素材，用「导入已有视频」可跳过通义万相，只需几十秒完成配音+字幕合成。

🏁 完整流程

  • 通义万相模式：选产品图 → 选模板 → 调参数(画质/时长/比例/BGM/TTS) → AI 自动 → 生成视频 → 等待 2-10 分钟 → 下载
  • 导入视频模式：上传视频 → 勾选/取消保留原声 → 填产品名 → AI 配乐+字幕 → 等待 30-60 秒 → 下载
