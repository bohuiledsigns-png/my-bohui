1.1 1500# Hairline Brushed Stainless Steel（拉丝不锈钢）

  材料起点: SUS304 / AISI 304 / 06Cr19Ni10
  供货状态: 2B 冷轧原板 → 单向拉丝处理
  厚度: 1.5mm（标准字壳）/ 1.2mm（轻薄型）/ 2.0mm（重型户外）

  拉丝工艺:
    - 砂带: 1500# 氧化铝（日本NCA或韩国DEERFOS）
    - 研磨方向: 单向直线，与字壳最长边平行
    - 研磨速度: 12-15 m/min（过快→发热变色，过慢→纹路过深）
    - 最终粗糙度 Ra: ≤0.4μm（JIS B0601触针法检测）
    - 纹路深度: 5-8μm | 纹路间距: 10-20μm

  为什么是1500#而不是800#或3000#:
    800# (Ra~0.8μm): 太粗糙，指纹收纳器，显脏。开快餐店可以用。
    1500# (Ra≤0.4μm): 有触感却不刮手，反光流动不刺眼。适合写字楼。
    3000# (Ra≤0.1μm): 接近镜面，指纹收集器，生产慢三倍。只适合奢侈品店。

  博汇话术:
  "1500# is the Goldilocks zone of metal — tactile enough to feel
   premium, smooth enough to wipe clean with a breath."


1.2 R3 Seamless Radial Corners（R3无缝精密R角）

  定义: 字壳转角圆弧半径 3mm，激光切割圆弧插补一次性成型。
  圆角类型: G2连续（曲率连续，无拐点）
  检测: R规(半径规) + 20×投影仪
  验收: 圆弧与直线段切点肉眼不可辨，手摸无顿挫感

  "R3不是半径，是傲慢"的工程本质:
    < 2mm: 锐角 → 应力集中 → 电镀/喷涂层易开裂
    > 8mm: 太圆 → 字壳失去精密刚毅感
    R3: 黄金平衡点 → 精密 × 结构 × 效率

    <3mm时必须降速50%才能切割 → 效率腰斩 → 成本翻倍
    所以R3不仅是一个尺寸——这是一个制造业哲学选择:
    我不降速、不加价、不减配——R3就是标准。

  博汇话术:
  "Other shops charge $80 per letter for corner sanding.
   We charge zero. Because a GLOWFORGE edge doesn't need sanding.
   R3 isn't a radius. It's a middle finger to the competition."


1.3 37° Light Refraction Chamfering（37°精密折射倒角）

  定义: 亚克力芯板每层阶梯的反射面倾角为 37°
  科学依据: PMMA（亚克力）在 6500K LED 光谱下的临界全反射角 (TIR)

  为什么是37°不是36°不是38°:
    36°: 光线跳过出口打到下一层 → 产生暗带
    37°: 入光效率 >96%，二次折射后出光均匀度最优
    38°: 光线反弹回LED灯带 → 能量浪费，红外热积累

    行业内多数工厂用 35-40° 随意倒角——因为他们不懂光学。
    37° 是我们用光谱仪测了 200 组数据锁定的唯一最优解。

  工程参数:
    步进深度: 0.3mm/step
    步进数: 32 steps（标准12mm亚克力板）
    总雕刻深度: 9.6mm（预留2.4mm基底强度）
    亮度分层: 32级（实测抽检，5帧数据稳定检出）
    出光均匀度: ±8%（行业标准 ±20%）

  博汇话术:
  "Why 37 degrees? Because physics said so.
   At 36°, dark band. At 38%, wasted energy.
   37° is the lock. The sweet spot.
   The difference between a backlit sign and a letter
   that holds light like a chalice."


1.4 Laser-Seamed Hermetic Welding（激光无缝气密焊接）

  定义: 激光搭接焊，字壳各面的永久性连接工艺。

  参数:
    对接缝间隙: <0.05mm（CNC折弯后的装配精度）
    焊接速度: 30-40 mm/s
    焊缝宽度: 0.3-0.5mm
    熔深: 0.3-0.5mm（不完全穿透 → 保持背面原色）
    后处理: 零打磨

  四种焊接工艺终极对比:
  ┌──────────────┬──────────┬─────────┬───────────┬────────────┐
  │               │ 氩弧焊    │ 锡焊    │ 电阻焊    │ 激光焊接    │
  ├──────────────┼──────────┼─────────┼───────────┼────────────┤
  │ 焊点大小      │ 3-5mm   │ 2-4mm  │ 1-2mm    │ <0.5mm    │
  │ 需打磨        │ 是       │ 是      │ 否        │ 否         │
  │ 色差          │ 明显     │ 轻微    │ 轻微      │ 无         │
  │ 拉丝连续性    │ 断       │ 基本    │ 基本      │ 完全连续   │
  │ 单字耗时      │ 8-12min │ 5-8min │ 2-3min   │ <1min     │
  │ 无需技能      │ 高级焊工 │ 中级   │ 操作员    │ 操作员     │
  └──────────────┴──────────┴─────────┴───────────┴────────────┘

  "Hand us a magnifying glass and a torch.
   You still won't find the seam.
   It's not hidden — it was never there."

2.1 Dual-Channel Independent Outline & Fill Control

  系统将"一个字"拆分为两套完全独立的神经系统:

  Channel A — Outline Controller（轮廓层）:
    - 控制字壳边缘的光效
    - 独立PWM @ ≥200Hz（零闪烁感）
    - 亮度 0-100% | 呼吸频率 0.5-2.0Hz | 相位偏移 0-360°
    - 负责"形体宣告"——"我在这里，长这样"

  Channel B — Fill Controller（填充层）:
    - 控制亚克力芯板的内部光效
    - 独立PWM + 软启动电路 (200ms)
    - 步进深度 0.1-0.5mm | 色温 2700-6500K
    - 负责"氛围营造"——"我有温度，我在呼吸"

  实测参数（基于5帧1280×720采样验证）:
    轮廓光像素: 帧1=6,681 → 帧5=8,345（+24.9%，单调递增锁定）
    填充光像素: 帧1=372,472 → 帧5=374,041（呼吸波动）
    轮廓/填充比: 1:53
    呼吸频率: ~0.8Hz（舒适慢闪）
    呼吸振幅: 3.0%（7.7/255）
    幻彩占比: 83.6%-97.7%（RGB分离度 >50阈值）


2.2 3000h Photometric Decay Resistance（3000小时光衰测试）

  定义: LED + 亚克力系统在持续点亮条件下光输出随时间衰减率。

  测试标准: LM-80 / QUV-A (ASTM D4329)

  对比数据:
  ┌───────────────────┬──────────────┬──────────────┐
  │                    │ 中国普通出口级 │ GLOWFORGE系统 │
  ├───────────────────┼──────────────┼──────────────┤
  │ 亚克力类型        │ Extruded     │ Cast (浇铸)  │
  │ UV稳定剂浓度      │ 1.5%         │ 2.5%          │
  │ 3000h光衰         │ 11%          │ 2.3%          │
  │ 8000h光衰（推算） │ 28%          │ 5.8%          │
  │ 3年亮度保持       │ ~65%         │ ~92%          │
  └───────────────────┴──────────────┴──────────────┘

  差异来源:
    1. 浇铸级PMMA分子链比挤出级长3倍——UV破坏更慢
    2. UV稳定剂浓度是行业标准的1.67倍
    3. 37° TIR折射角使红外热积累最小化（IR吸收率<2%）
    4. LED选用低UV输出型（4000-6500K光谱峰值偏移可见光波段）

  博汇话术:
  "Your current sign loses 11% brightness in 4 months.
   You won't notice because it happens so slowly.
   But the sign next door, installed on the same street
   at the same time? It's still at 97.7%.
   After a year, the gap is visible from across the road.
   After three years, yours looks like a backup light."

3.1 Raceway Mounting System（线槽式安装）

  为什么海外市场为Raceway疯狂:
    海外人工 $120-200/h。传统安装: 现场逐个挂字+走线+测试 = 3-4h×2电工
    博汇预制Raceway: 工厂预穿线+分线端子+快插头 = 30min×1普通工人
    一个项目省 $1,000-2,000 安装费

  博汇Raceway标准:
    材质: 6063-T5铝合金，壁厚2mm+
    表面: 阳极氧化银白 / 粉末喷涂（与字壳同色）
    内部: 预穿线 + 分线端子 + MeanWell电源固定架
    防水: 盖板配EPDM密封条，IP65
    接口: 快插式航空插头（一插即通电）
    承重: 每米挂40kg+（满足绝大多数Channel Letters）
    配件: 水平仪定位支架（老外不需要专业工具也能装直）

  "A raceway is the most boring-sounding innovation in signage.
   It's also the one your accountant will thank us for."

4.1 ADA-Compliant Braille Signage（盲文标识）

  法规: 美国残疾人法案 (ADA)，适用于所有公共建筑
  核心参数:
    - 盲文点 (Braille Dots): 圆顶，高 0.6-0.8mm
    - 字符高度: ≥5/8"（16mm），大写
    - 对比度: 字符与背景反射率差 >70%
    - 安装: 文字下方必须配盲文，无尖锐边缘
    - 高度: 中心线距地面 48-60"（1220-1525mm）

  博汇工艺:
    材料: 304不锈钢 / 亚克力 / HDU
    盲文: 激光蚀刻——永久不脱落（同行用粘贴式——会掉）
    圆顶精度: 0.7mm ± 0.05mm（手指闭眼可辨识）
    文字: 1500#拉丝不锈钢或3D亚克力立体字
    安装: Pin Mount悬浮或Flush Mount平贴

  "Other factories glue braille. We etch it into the metal.
   ADA inspectors don't pass glued dots. Ours? First try."


4.2 HDU Board 3D Carving（高密度聚氨酯板3D雕刻）

  参数:

    户外寿命: 15年+（木头户外3年）

  应用场景:

  "木头好看，但3年就烂了。HDU好看，15年不烂。

4.3 ACM Panel（铝塑板—户外背板之王）

  结构: 0.3-0.5mm铝合金面板 × 2 + PE/PVDF芯层
  厚度: 3-6mm（招牌）/ 6-10mm（幕墙）
  耐候: PVDF涂层 20年+ 户外暴晒
  抗风: 140 mph（225 km/h）
  加工: CNC雕刻 / 折弯 / 开槽 / 钻孔

  "99%的供应商把ACM当利润中心。我们当它沉默零件——不出问题就是最高标准。"


4.4 Push-Through Acrylic Sign（亚克力透字招牌）

  工艺: 在ACM背板上开精密槽孔 → 亚克力字从背面"推"出5-15mm
  立体感来源: 字厚度 + 凸起高度 + 背板的阴影间隙
  槽孔公差: <0.2mm（否则字会晃动或卡不进去）

  "A push-through sign is the gentleman's channel letter.
   No visible wiring. No raceway. No excuses."


4.5 Interchangeable Insert System（可替换插槽系统）

  定义: 室内导视牌的面板可徒手更换文字内容
  适用: 写字楼楼层索引 / 医院科室牌 / 学校教室牌
  结构: 铝合金边框 + 亚克力透明面板 + 可替换插卡纸/UV打印片
  特点: 换内容不需要拆墙、不需要断电、不需要工具

  "Most directories are 'install and pray nothing changes.'
   Ours? Your tenant moves out on Friday,
   new name goes in on Monday. Zero wall damage."
