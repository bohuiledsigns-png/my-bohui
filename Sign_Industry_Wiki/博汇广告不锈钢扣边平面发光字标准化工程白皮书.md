1.1 316# Marine-Grade Stainless Steel（航海级不锈钢）
  ── 决战沿海盐雾 · 赌场 · 豪华游艇码头

  材料: SUS316 / AISI 316 / 06Cr17Ni12Mo2
  核心优势: 含 2-3% 钼（Mo），抗氯离子点蚀能力是 304 的 3 倍
  适用场景:
    - 距海岸线 5km 以内的户外招牌（盐雾腐蚀是室内环境的 20 倍）
    - 游艇码头、海滨度假村、赌场、泳池会所
    - 食品加工厂外墙（316# 耐酸性气体侵蚀）
    - 中东/东南亚沿海项目（波斯湾/南海的高盐雾+高温地狱组合）
  成本: 304# 的 1.8-2.2 倍
  辨别: 光谱仪检测 Mo 元素含量 > 2%

  博汇话术:
  "316# costs twice as much as 304#. You know when you need it?
   When the ocean is 500 meters away and the client says
   'I want this to outlast the building.'"
  "Your competitor bids 304# for that beachfront project.
   We bid 316#. In 5 years, theirs has pitting corrosion.
   Ours still looks like the day it was hung."


1.2 304# Architectural-Grade Stainless Steel（建筑级不锈钢）
  ── 中流砥柱 · 90% 项目的默认选择

  材料: SUS304 / AISI 304 / 06Cr19Ni10
  供货状态: 2B 冷轧原板 → 1500# 单向拉丝
  厚度: 1.2mm（室内轻薄字壳）/ 1.5mm（标准门头）/ 2.0mm（户外重型）
  适用场景:
    - 99% 的城市户外门头招牌
    - 商业综合体导视系统
    - 办公室/写字楼室内标识
    - 只要不直接接触海水或强酸，304# 就是最优解
  屈服强度: ≥205 MPa
  抗拉强度: ≥520 MPa

  博汇话术:
  "304# is the Toyota Hilux of stainless steel.
   It's not the flashiest. It's not the cheapest.
   But it will start every morning for 20 years
   without complaining."
  "如果你不在海边开餐厅，304# 是你最聪明的选择。
   多花的每一分钱买的是 20 年不生锈的确定性。"


1.3 201# Interior-Only Stainless Steel（仅限室内干燥环境）
  ── 廉价替代品 · 仅用于不会淋到雨的地方

  材料: SUS201 / AISI 201
  成分差异: 镍（Ni）含量 3.5-5.5%（304# 为 8-10.5%），锰（Mn）替代
  核心短板:
    - 无钼（Mo）→ 抗点蚀能力极弱
    - 户外 6-12 个月开始生锈（尤其沿海）
    - 紫外线下加速变色
  允许使用场景:
    - 室内干燥区（商场中庭、办公楼大堂内墙）
    - 短期限时项目（展览/快闪店/临时促销）
    - 客户的 Budget 极度有限且明确告知"只挂一年"
  绝对禁止: 任何户外、半户外、近海、高湿环境

  博汇话术:
  "201# is not 'budget-friendly.' It's 'we-told-you-so-friendly.'
   Use it indoors. Use it dry. Don't say we didn't warn you."
  "如果你把 201# 挂到外墙上——你在跟天打赌。
   博汇不会阻止你，但我们会让你签一份免责书。
   上面写着：'我知道这是 201#，我知道它会在一年内生锈。'"

2.1 Laser-Cut（激光切割——起点决定终点）

  设备: 500W-1kW 光纤激光切割机（GLOWFORGE 系统配套）
  精度: ±0.1mm（头发丝级）
  切口宽度: 0.12-0.15mm（竞品 CNC 铣削 3.3mm，是激光的 22 倍）
  辅助气: 氮气 @ 0.8MPa（防止切面氧化发黑）
  焦点: 材料表面（0 偏移）
  热影响区 HAZ: <0.05mm

  对比 CNC 铣削:
    CNC 切口 3.3mm → 需要二次打磨填缝 → 每字壳 +8-15min
    激光切口 0.12mm → 出光口已火焰抛光 → 零后处理


2.2 Seamless Micro-Welding（无缝微焊接）

  定义: 字壳面板与围边的永久性连接工艺
  类型: Laser Lap Welding（激光搭接焊）
  对接缝间隙: <0.05mm（CNC 折弯后的装配精度）
  焊接速度: 30-40 mm/s
  焊缝宽度: 0.3-0.5mm
  熔深: 0.3-0.5mm（不完全穿透 → 保持背面原色）
  后处理: 零打磨

  四种焊接工艺对比:
  ┌──────────────┬──────────┬─────────┬───────────┬────────────┐
  │               │ 氩弧焊    │ 锡焊    │ 电阻焊    │ 激光焊接   │
  ├──────────────┼──────────┼─────────┼───────────┼────────────┤
  │ 焊点大小      │ 3-5mm   │ 2-4mm  │ 1-2mm    │ <0.5mm    │
  │ 需打磨        │ 是       │ 是      │ 否        │ 否         │
  │ 色差          │ 明显     │ 轻微    │ 轻微      │ 无         │
  │ 拉丝连续性    │ 断       │ 基本    │ 基本      │ 完全连续  │
  │ 单字耗时      │ 8-12min │ 5-8min │ 2-3min   │ <1min     │
  └──────────────┴──────────┴─────────┴───────────┴────────────┘

  博汇话术:
  "A 3mm weld bead screams 'I was made in a hurry.'
   A 0.3mm laser seam whispers 'I was made with intention.'
   Your client will never see the difference.
   But they'll feel it when they run their finger across the letter."


2.3 无缝打磨抛光（Seamless Grinding & Polishing）

  只有氩弧焊和锡焊需要打磨——激光焊接不需要。
  但对于少数指定镜面抛光（Mirror Finish）的项目:

  工艺链:
    #400 粗磨 → #800 中磨 → #1500 细磨 → 布轮抛光
  每字壳耗时: 15-25min
  最终 Ra: ≤0.05μm（镜面级）

  博汇标准:
    镜面抛光  = 客户明确指定 + 额外工期 + 额外费用
    拉丝表面  = 默认标准（1500#，Ra≤0.4μm）


2.4 1500# Hairline Brushed Finish（1500# 高级拉丝）

  砂带: 1500# 氧化铝（日本 NCA 或韩国 DEERFOS）
  研磨方向: 单向直线，与字壳最长边平行
  研磨速度: 12-15 m/min
  最终 Ra: ≤0.4μm
  纹路深度: 5-8μm
  纹路间距: 10-20μm

  三个等级的区别:
    800# (Ra~0.8μm): 太粗糙，指纹收纳器，显脏
    1500# (Ra≤0.4μm): 有触感不刮手，反光流动不刺眼 ✅ 博汇标准
    3000# (Ra≤0.1μm): 接近镜面，指纹收集器，生产慢三倍


2.5 电镀金钛工艺（PVD Gold / Titanium Coating）

  定义: 物理气相沉积（PVD），在不锈钢表面镀一层金色或钛色
  厚度: 0.3-1.0μm
  硬度: 2,000-3,000 HV（是不锈钢本身的 5-8 倍）
  耐候: 户外 5-8 年不褪色（电镀工艺的好差决定最终寿命）

  适用场景:
    - 高端奢侈品店门头（金色字壳配暖色灯光）
    - 酒店大堂导视牌（香槟金/玫瑰金）
    - 赌场/会所（土豪金）
    - 特殊品牌色定制（黑色钛、枪色、青铜色）

  博汇标准:
    博汇不推荐大面积电镀——因为电镀层一旦破损无法修复。
    如果客户坚持，我们建议只在局部装饰面做 PVD，
    主体结构保持 1500# 拉丝本色。

3.1 扣边宽度动态标准（Trim-Cap Width by Letter Size）

  定义: 扣边（Trim-Cap）= 围绕亚克力面罩的不锈钢/铝C型包边条

  ┌──────────────────────┬──────────────────┬─────────────────────┐
  │ 字高度                 │ 扣边宽度           │ 适用场景             │
  ├──────────────────────┼──────────────────┼─────────────────────┤
  │ < 300mm (1ft)        │ 2mm              │ 室内导视、小门牌     │
  │ 300-600mm (1-2ft)    │ 3mm              │ 标准门头字           │
  │ 600-1000mm (2-3ft)   │ 5mm              │ 大型门头、建筑标识   │
  │ 1000-1500mm (3-5ft)  │ 8mm              │ 大门头、Pylon面板    │
  │ > 1500mm (>5ft)      │ 10mm             │ Monument级巨型字     │
  └──────────────────────┴──────────────────┴─────────────────────┘

  为什么扣边宽度必须随字大小递增:
    字越大 → 面板面积越大 → 风载越大 → 扣边需要越宽以抵抗变形
    扣边不仅是装饰——它是亚克力面罩的结构固定系统


3.2 超 1 米大字的围边加强标准

  定义: 围边（Return / Sidewall）= 字壳的侧面深度，决定字体的立体厚度

  标准围边厚度: 40-80mm（一般扣边平面发光字）
  特殊情况:
    - 字高度 > 1m 或 宽度 > 1.2m: 围边必须加宽到 80-120mm
    - 字高度 > 1.5m: 围边内部需加横向加强筋（Stiffener Rib）
    - 字高度 > 2m: 围边必须加厚材料（1.5mm→2.0mm）+ 内部十字加强筋

  为什么要加宽/加厚围边:
    Wind Load（风载）对大面积平面的吸力:
    - 1m² 面积在 180km/h 风速下承受约 2,500N（≈255kg）的拉力
    - 5m² 的巨型字承受超过 1,200kg 的拉力
    - 如果围边不够宽、材料不够厚，字壳会在暴风中撕裂

  博汇标准:
    "Any letter with face area > 1m² gets structural reinforcement.
     Not negotiable. Not optional."


3.3 亚克力面罩厚度动态标准（Acrylic Face Thickness by Size）

  定义: 覆盖在扣边内的发光面板，光从背后穿透

  ┌─────────────────┬───────────────┬──────────────────────────┐
  │ 字高度             │ 亚克力厚度      │ 说明                      │
  ├─────────────────┼───────────────┼──────────────────────────┤
  │ < 300mm         │ 3mm           │ 室内小字，低风载风险     │
  │ 300-600mm       │ 4-5mm         │ 标准门头，通用厚度       │
  │ 600-1000mm      │ 5-6mm         │ 大字需更强刚性防凹陷     │
  │ 1000-1500mm     │ 6-8mm         │ 超大字，防风压弯曲       │
  │ > 1500mm        │ 8mm+          │ 工程定制，内部加强筋     │
  └─────────────────┴───────────────┴──────────────────────────┘

  "A 1.5m letter with 3mm acrylic is a drumhead waiting for wind.
   We calculate the acrylic thickness based on face area × wind load.
   Not by whatever we have in stock."

4.1 Interlocking 天地盖无螺丝工艺（Tool-Free Backplate System）

  定义: 字壳背部封板采用嵌入式卡扣设计，无需螺丝固定

  结构:
    - 字壳围边内侧预冲卡槽（间距 150-200mm）
    - PVC 封底板边缘匹配卡槽形状
    - 安装时从侧边滑入 → 卡扣自动锁紧
    - 拆卸时用扁口螺丝刀轻轻一撬 → 卡扣释放

  优势:
    - 零螺丝 → 无生锈风险
    - 零工具 → 现场安装/维护不需要电钻
    - 防水密闭 → 卡扣配合 EPDM 密封条，IP54 级
    - 维修便利 → 徒手拆背板换 LED 灯带，无需拆字壳

  对标竞品:
    同行: 背板用自攻螺丝锁死 → 螺丝生锈 → 拆的时候滑丝 → 整个字报废
    博汇: Interlocking卡扣 → 10秒开/10秒关 → 永久不生锈 → 可反复拆卸


4.2 High-Hardness PVC Backplate（高硬度 PVC 封底）

  材料: 硬质 PVC 板（PVC Rigid Sheet）
  厚度: 3-5mm（标准）/ 5-8mm（大尺寸加强）
  特性:
    - 绝缘（不导电）→ 零短路风险
    - 防水（不吸水）→ 户外潮湿环境零变形
    - 阻燃（UL94 V-0 级）→ 符合海外消防验收
    - 轻量（比重 1.4，是不锈钢的 1/5）→ 减轻整体承重

  对比竞品:
    同行用镀锌铁皮做背板:
    - 导电（短路风险）→ 需要额外绝缘垫片
    - 生锈（2-3年边缘开始锈蚀）→ 客户投诉
    - 重（比 PVC 重 5 倍）→ 安装更费力
    博汇用 PVC: 轻、绝缘、防水、阻燃——四个维度全面碾压


4.3 12V Low-Voltage Waterproof LED（12V 低压防水 LED 灯带）

  参数:
    电压: 12V DC（低压安全，海外电工随便接）
    防水等级: IP65（防尘防喷水）/ IP67（短时浸水，选配）
    LED 芯片: SMD 2835 或 5050
    灯珠密度: 60-120 珠/米
    色温选项: 2700K（暖黄） / 3000K（暖白） / 4000K（自然白）
