1.1 Wayfinding Signage（室内导视系统）

  定义: 建筑室内空间中帮助人们定位、导航的标识系统。
  层级: 
    Level 1 — 总平面图 (You Are Here Map)
    Level 2 — 方向指引 (Directional / Directory)
    Level 3 — 区域标识 (Room ID / Department ID)
    Level 4 — 信息标识 (Hours / Policy / Amenities)

  核心工艺:
    - 底板: ACM (铝塑板) / HDU (高密度聚氨酯) / 拉丝不锈钢
    - 表面: 烤漆 (Powder Coating) / 阳极氧化 (Anodizing) / 丝印 (Screen Print)
    - 文字: 3D亚克力立体字 / 金属蚀刻 / UV数码直印
    - 安装: Flush Mount (平贴) / Pin Mount (悬浮) / Raceway (线槽)

  博汇壁垒:
    同行做导视 = 丝印一块铝板，贴上去。
    博汇做导视 = 拉丝不锈钢底板 + R3圆角亚克力立体字 + 
                  Pin Mount悬浮安装 + 阴影间隙精确到2mm。


1.2 ADA-Compliant Signage（美国残疾人法案合规标识）

  定义: 美国法律要求的盲文标识系统，用于公共建筑。
  核心参数:
    - 盲文 (Braille): 圆顶 (Dome Shape)，高 0.6-0.8mm
    - 字符高度: 最低 5/8" (16mm)，大写
    - 对比度: 字符与背景的反射率差 > 70%
    - 安装高度: 中心线距地面 48"-60" (1220-1525mm)
    - 可触达: 文字下方必须配盲文，不可有尖锐边缘

  材料选择:
    - 亚克力 (Acrylic): 外观现代但耐刮差
    - 不锈钢 (Stainless): 博汇推荐——1500#拉丝 + 激光蚀刻盲文
    - HDU (高密度聚氨酯): 复古风格，可CNC雕刻
    - 复合材料 (ACM): 经济型，适用大面积

  博汇壁垒:
    大多数中国工厂做ADA = 丝印上文字 + 贴塑料盲文点（会掉）。
    博汇做ADA = 304不锈钢底板 + 激光蚀刻盲文点（永久不脱落）。
    盲文点的圆顶高度 0.7mm ± 0.05mm — 用手指闭眼摸可辨识。

2.1 Pylon Sign / Monument Sign（立柱招牌与地标招牌）

  Pylon Sign:
    - 单柱或双柱支撑，高度 6-20m
    - 常见: 高速公路旁、加油站、购物中心入口
    - 结构: 钢构基础 + 铝板饰面 + LED照明 + 亚克力面板
    - 风载: AS1170.2 (澳洲) / ASCE 7 (美国) 标准
    - 基础: 混凝土配筋，深度 1.5-3m（视土壤和风区而定）

  Monument Sign:
    - 矮宽式地标，高度 1-3m
    - 常见: 商业园区入口、高端住宅社区
    - 饰面: 石材/砖石/铝板/Stucco
    - 特点: 需要与周围景观协调，设计师审美要求高

  核心结构参数:
    - 钢结构: 热镀锌方管/工字钢，Q235B或Q345B
    - 饰面: 2-3mm 铝板，氟碳喷涂 (PVDF) 或 粉末喷涂
    - 照明: LED泛光灯 / 背打光 / 内透光
    - 电气: IP65防水接线盒 + MeanWell电源 + 漏电保护
    - 基础: C30混凝土，地脚螺栓 M24-M36


2.2 抗风载结构设计（海外工程核心痛点）

  为什么老外最怕这个:
    澳洲 cyclone / 美国 hurricane / 英国 storm — 风灾索赔是招牌行业
    最大的法律风险。一个不抗风的Pylon Sign被吹倒 = 赔偿$50K起。

  博汇标准:
    - 设计风速: 50m/s (180km/h) — 覆盖Category 3飓风
    - 安全系数: 1.5 (行业最低1.2)
    - 焊缝: 全熔透焊接 + 超声波探伤抽检
    - 螺栓: 304不锈钢，防松垫片 + 扭矩标记
    - 计算书: 提供中英文版结构计算书（老外工程师可签字）

  同行 vs 博汇:
    同行: "2"铁管焊接一下就行了，老外现场自己装。
    博汇: "有限元风载分析 (FEA) → 结构计算书 → 工厂预制 → 
           模块化发货 → 现场螺栓拼接 → 质保10年结构。


2.3 Channel Letters（户外发光字）

  Front-Lit（正面发光）:
    - 字壳: 铝/不锈钢折弯 + Trim Cap包边
    - 面罩: 亚克力 3-5mm
    - LED: 防水型 SMD 2835/5050，IP65
    - 典型厚度: 60-80mm

  Reverse / Halo-Lit（背发光）:
    - 字壳: 不锈钢/铝（背面开放）
    - LED: 打在墙面或背板上
    - 效果: 字本身暗，墙上有一圈光晕
    - 典型安装: Pin Mount (离墙20-50mm)

  Open-Face（开放式）:
    - 字壳: 无面罩，内部LED裸露
    - LED: 高亮度防水型，每个字独立
    - 效果: 每个字是一盏独立的灯
    - 博汇推荐: 配合GLOWFORGE双通道控光


2.4 户外大门头 Raceway 安装系统

  为什么Raceway是大门头的灵魂:
    海外人工 $120-200/h。现场逐个挂字+走线+测试 = 4h×2电。
    博汇预制Raceway线槽 = 30min×1普通工人。
    一个项目省 $1,200-2,000 安装费。

  博汇Raceway标准:
    - 材质: 6063-T5铝合金，壁厚2mm+
    - 表面: 阳极氧化银白 / 粉末喷涂（与字壳同色）
    - 内部: 预穿线 + 分线端子 + MeanWell电源固定架
    - 防水: 盖板配EPDM密封条，IP65
    - 接口: 快插式航空插头（一插即通电）
    - 承重: 每米可挂40kg+ (满足绝大多数Channel Letters)

3.1 ACM (Aluminum Composite Material) 铝塑板

  结构: 0.3-0.5mm 铝合金面板 × 2 + PE/PVDF芯层
  厚度: 3-6mm (招牌) / 6-10mm (幕墙)
  耐候: PVDF涂层 20年+ 户外
  抗风: 140 mph (225 km/h)
  加工: CNC雕刻 / 折弯 / 开槽 / 钻孔
  品牌推荐: Reynobond / Alucobond / 博汇自有供应

  博汇话术:
  "ACM是招牌的龙骨。99%的供应商把ACM当利润中心。
   我们把它当沉默零件——不出问题就是最高标准。"


3.2 HDU Board (High-Density Urethane) 高密度聚氨酯

  密度: 15-30 lb/ft³
  耐候: 户外15年+（木头3年）
  加工: CNC雕刻任意3D形状
  涂装: 需要专用聚氨酯底漆
  场景: 复古街区招牌 / 高端住宅 / 历史保护区

  博汇话术:
  "木头好看但3年烂。HDU好看，15年不烂。
   这就是为什么一个CNC雕刻的木纹替代品卖得比真木头还贵。"


3.3 Optically Clear Cast Acrylic（光学级浇铸亚克力）

  透光率: ≥92% @ 550nm
  UV稳定剂: 2.5% （行业标准 1.5%）
  黄变测试: 3000h QUV-A, ΔE ≤ 1.5（肉眼不可辨）
  加工: 激光切割 / CNC雕刻 / 火焰抛光 / 步进雕刻

  行业内幕:
  "挤出级便宜30%，一年就黄。光学级贵30%，三年后跟新的一样。
   这个三年代价，你想省谁的钱？"


3.4 MeanWell Power Supply（明纬电源）

  品牌: 台湾明纬 (MeanWell)，全球LED驱动电源第一品牌
  型号: LRS-150 / HLG-240H 等
  抗浪涌: 6kV
  工作温度: -30°C to +70°C
  MTBF: >300,000 小时（约34年连续运行）
  认证: UL / CE / FCC / RoHS

  博汇话术:
  "MeanWell是生意人的睡眠保证书。
   你的招牌嗡嗡响15年不坏，客户忘了招牌的存在。
   这就是最好的招牌。"


3.5 304 Stainless Steel（304不锈钢）

  牌号: SUS304 / AISI 304 / 06Cr19Ni10
  厚度: 1.2-2.0mm (字壳推荐 1.5mm)
  表面: 2B原板 → 1500#单向拉丝
  Ra: ≤0.4μm
  屈服强度: ≥205 MPa
  抗拉强度: ≥520 MPa
  焊接: 激光无缝焊接，熔融区<0.1mm
