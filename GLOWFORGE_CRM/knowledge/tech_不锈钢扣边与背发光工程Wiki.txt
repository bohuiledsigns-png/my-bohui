1.2 激光切割与无缝焊接

  Laser-Cut（激光切割）:
    精度: ±0.1mm | 切口: 0.12-0.15mm（CNC的22倍精细度）
    氮气辅助 @0.8MPa | HAZ <0.05mm | 出光口即火焰抛光

  Seamless Micro-Welding（无缝微焊接）:
    类型: Laser Lap Welding | 焊缝: 0.3-0.5mm
    对接缝间隙: <0.05mm | 零打磨 | 拉丝连续跨过焊缝

  四种焊接工艺对比:
    氩弧焊 3-5mm → 需打磨 → 色差明显 → 拉丝断
    锡焊 2-4mm → 需打磨 → 轻微色差
    电阻焊 1-2mm → 零打磨 → 基本连续
    激光焊 <0.5mm → 零打磨 → 零色差 → 拉丝完全连续 ✅


1.3 1500# Hairline 拉丝与电镀钛金

  拉丝标准:
    砂带: 1500# 氧化铝（日本NCA/韩国DEERFOS）
    方向: 单向直线，与字壳最长边平行
    Ra: ≤0.4μm | 纹路深度: 5-8μm | 间距: 10-20μm

  电镀钛金 PVD:
    颜色: 黑色钛 / 玫瑰金 / 香槟金 / 青铜色
    厚度: 0.3-1.0μm | 硬度: 2000-3000 HV
    耐候: 户外5-8年不褪色
    顺序: 焊接→打磨→拉丝→PVD（最后一道工序）

2.1 扣边宽度动态标准（Trim-Cap Width by Letter Size）

  ┌──────────────────────┬──────────────────┬─────────────────────┐
  │ 字高度                  │ 扣边宽度           │ 说明                   │
  ├──────────────────────┼──────────────────┼─────────────────────┤
  │ < 300mm              │ 2mm              │ 室内小字               │
  │ 300-600mm            │ 3mm              │ 标准门头 ← 最常见      │
  │ 600-1000mm           │ 5mm              │ 大型门头               │
  │ 1000-1500mm          │ 8mm              │ 大字，加宽抗风         │
  │ > 1500mm             │ 10mm             │ 巨型字，结构加强       │
  └──────────────────────┴──────────────────┴─────────────────────┘

  原理: 字越大 → 风载越大 → 扣边必须越宽以抵抗亚克力面板变形
  1m²面在180km/h风速下承受约255kg拉力


2.2 超1米大字的围边加强（Return/Sidewall Reinforcement）

  标准围边厚度: 40-80mm

  超1米特殊情况:
    字高 > 1m 或宽 > 1.2m:  围边加宽到 80-120mm
    字高 > 1.5m:             内部加横向加强筋
    字高 > 2m:               材料加厚到 2.0mm + 十字加强筋

  背发光字围边（英国大单规范）:
    标准厚度: 50-60mm（容纳10-15mm水晶底板 + LED模组 + 排线）
    围边深度不足 → 水晶底板装不进去 → 光晕效果打折

3.1 水晶底座 Acrylic Crystal Base（10-15mm光学级高透亚克力）

  材料: Optically Clear Cast Acrylic（浇铸级）— 严禁挤出级
  透光率: ≥92% @ 550nm
  UV稳定剂: 2.5%（行业标准1.5%，我们是1.67倍）
  UV测试: 3000h QUV-A, ΔE ≤ 1.5（肉眼不可辨黄变）

  厚度选择:
    字高 < 300mm:    10mm 匀光足够
    字高 300-600mm:  10-12mm ← 英国单标准
    字高 600-1m:    12-15mm
    字高 > 1m:      15mm+ 或特厚定制

  360°匀光原理:
    ① 10-15mm厚板 = 天然导光体，光线经多次内部反射后出射
    ② 浇铸级PMMA分子链宽密，光线传播均匀
    ③ 背板微磨砂处理（Ra 1-2μm）→ 消除最后热点
    → 彻底消除传统背发光的刺眼光斑


3.2 不锈钢天地盖无螺丝工艺（Interlocking Tool-Free Backplate）

  结构:
    字壳由两件套组成:
    - 上盖（Face Side）: 不锈钢面板 + 围边，激光焊接一体
    - 下盖（Back Side）: 不锈钢或PVC封板，Interlocking卡扣

  安装流程:
    ① 排LED模组 → 接线 → 穿过Cable Outlet
    ② 水晶底板嵌入围边凹槽 → 硅酮密封胶
    ③ 天地盖从底部滑入 → 卡扣自动锁紧
    ④ PG防水锁头锁死出线孔

  维护流程:
    ① 扁口螺丝刀轻撬卡扣释放
    ② 天地盖滑出 → 露出内部组件
    ③ 换LED/换电源/清理 → 10秒
    ④ 卡扣盖回去 → 完成

  "同行用螺丝锁死背板→螺丝生锈→滑丝→整个字报废。
   博汇天地盖→10秒开10秒关→永久不生锈→可反复拆卸。"


3.3 电缆出线孔 Cable Outlet

  孔径: 8-12mm（根据线束直径）
  位置: 字壳底部中间偏后（安装后不可见）
  防护: PG9/PG11防水电缆接头 + EPDM橡胶圈 → IP65
  密封: 锁头内侧垫橡胶圈 → 拧紧后防尘防喷水

  同行:
    "出线孔 = 进水孔。用胶布随便缠一下。"
  博汇:
    "PG级防水锁头。IP65。你的招牌防水寿命比建筑还长。"

4.1 12V Low-Voltage Waterproof LED Module（12V低压防水LED模组）

  类型: 背发光专用高亮模组 / 扣边字标准灯带
  电压: 12V DC（低压安全，全球通用）
  防水: IP65（标准）/ IP67（浸水级选配）
  芯片: SMD 2835
  排布: 轮廓内侧一圈密集排布 + 填充区域均匀分布（间距80-120mm）
        转角处加密（转角最容易出现暗区）

  色温选项:
    2700K 暖黄   3000K 暖白 ← 英国单/餐厅最常用
    4000K 自然白 5000K 冷白
    6500K 日光白 RGB幻彩（选配）

  扣边字排布:
    灯带贴在字壳底部，向上照亮亚克力面罩
    每100mm²一个模组，确保面罩均匀发光

  背发光（水晶底座）排布:
    模组朝后照射水晶底座，光线穿过底座打在墙上形成日冕
    轮廓侧加密（光晕饱满的关键）

4.3 T8 Multi-Channel Animation Controller（T8多通道动画控制器）

  定义: GLOWFORGE系统的8通道独立PWM控制器

  参数:
    8个独立输出通道
    每通道独立编程: 亮度 / 呼吸频率 0.5-2Hz / 相位 / 色温
    场景预设: 欢迎模式 / 夜晚模式 / 凌晨守夜模式 / 派对模式
    输入: 100-240V AC | 输出: 12V DC × 8通道
    通信: DMX512 / PWM / 0-10V调光
    尺寸: 220×150×50mm（可内置于Raceway线槽）

  典型编程:
    白天待机:  轮廓100% 填充0%   幻彩关闭
    黄昏迎宾:  轮廓80%  填充30→70% 低亮
    夜间标准:  轮廓70%  填充85%    关闭
    凌晨守夜:  轮廓10%  填充0%     关闭
    派对模式:  轮廓100% 填充100%+呼吸 全彩循环
