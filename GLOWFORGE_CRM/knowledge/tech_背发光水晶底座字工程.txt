1.2 激光切割实体表面（Laser-Cut Solid Face）

  设备: 500W-1kW 光纤激光切割机
  精度: ±0.1mm
  切口宽度: 0.12-0.15mm（竞品 CNC 铣削 3.3mm，激光的 22 倍精细）
  辅助气: 氮气 @ 0.8MPa（防氧化）
  热影响区 HAZ: <0.05mm

  对比 CNC 铣削:
    CNC 切口 3.3mm → 二次打磨填缝 → 每字壳 +8-15min
    激光切口 0.12mm → 出光口已火焰抛光 → 零后处理


1.3 围边与字壳厚度（Return Depth / Letter Thickness）

  定义: 围边（Return/Sidewall）= 字壳从正面到背面的垂直深度，
        决定了字的立体厚度和内部安装空间。

  ┌────────────────────────────────────────────────────────────────────┐
  │  标准厚度: 40-80mm                                                │
  │    - 40mm: 薄型，适合室内/空间受限处                              │
  │    - 60mm: 标准型，通用厚度，兼容内部LED排布                       │
  │    - 80mm: 加厚型，适合大字/需要更大内部空间安装水晶底座            │
  │                                                                     │
  │  英国订单典型值: 50-60mm                                           │
  │    - 足够容纳 10-15mm 水晶亚克力底板 + LED 模组 + 排线              │
  │    - 围边深度不足 → 水晶底板装不进去 → 光晕效果打折 → 客户不满     │
  └────────────────────────────────────────────────────────────────────┘


1.4 侧边精工工艺链（Sidewall Finishing Chain）

  选项 A — 无缝焊接 + 打磨抛光（Seamless Weld + Polish）
    适用: 镜面不锈钢 / 电镀钛金表面
    工艺: 激光焊接 → #400粗磨 → #800中磨 → #1500细磨 → 布轮抛光
    最终 Ra: ≤0.05μm（镜面级）
    耗时: 15-25min/字（额外计费）

  选项 B — 1500# Hairline 拉丝（Brushed Finish）✅ 博汇标准
    适用: 所有默认不锈钢表面
    砂带: 1500# 氧化铝（日本NCA/韩国DEERFOS）
    研磨方向: 单向直线，与字壳最长边平行
    研磨速度: 12-15 m/min
    最终 Ra: ≤0.4μm
    注意: 同一个字壳上所有可见面拉丝方向必须一致——不可交叉研磨

  选项 C — PVD 电镀钛金（PVD Titanium/Gold Coating）
    适用: 黑色钛 / 玫瑰金 / 香槟金 / 青铜色
    厚度: 0.3-1.0μm
    硬度: 2,000-3,000 HV（是不锈钢本身的 5-8 倍）
    耐候: 户外 5-8 年不褪色
    注意: PVD 是最后一道工序——必须在焊接/打磨/拉丝全部完成后才能做

  博汇标准:
    "The sidewall is the part your client sees first when they approach
     from an angle. If it's not finished to the same standard as the face
     — you're hiding something. We don't hide anything."

  基材: Optically Clear Cast Acrylic（光学级浇铸亚克力）

    透光率: ≥92% @ 550nm（绿光波段，人眼最敏感）
    UV稳定剂: 2.5%（行业标准 1.5% → 我们是标准的 1.67 倍）
    黄变测试: 3000h QUV-A, ΔE ≤ 1.5（肉眼不可辨）
    供应商: 三菱丽阳 / 璐彩特 / 申威达

  厚度选择:

  ├─────────────────┼────────────────┼──────────────────────────────┤

2.2 360° 完美匀光原理（360° Uniform Light Diffusion）

  背发光最常被投诉的问题: 光斑（Hotspots）——LED灯珠正对面特别亮，
  两个灯珠之间特别暗。原因: 亚克力太薄或者亚克力是挤出级。

  博汇水晶底板的匀光机制:
    1. 厚度: 10-15mm 的厚板本身就是天然的导光体——
       光线进入亚克力后经过多次内部反射才出射，天然打散光斑
    2. 浇铸级: 浇铸PMMA的分子链更宽更密，光线传播路径更均匀
    3. 背板处理: 水晶底板背面做微细磨砂处理（Ra 1-2μm）→
        进一步散射光线，消除最后一点热点

  "A 3mm extruded acrylic backplate is a tinted window.
   A 12mm cast acrylic crystal base is a light engine.
   The difference is the difference between 'lit' and 'illuminated.'"


2.3 防水防尘气密安装（Waterproof & Dustproof Sealing）

  水晶底板的安装方式:
    ① 亚克力底板轮廓按字壳内部形状 CNC 精密切割（公差 ±0.2mm）
    ② 底板边缘嵌入字壳围边内壁的凹槽（Depth Groove）
    ③ 接缝处打中性硅酮密封胶（Neutral Silicone Sealant）
    ④ 上盖板（不锈钢天地盖）锁紧 → 密封胶固化
    ⑤ 最终气密等级: IP54（防尘防溅水）/ 选配 IP65

  对比竞品:
    同行: 亚克力板直接贴在字壳背面 → 缝隙进水 → LED短路
    博汇: 凹槽嵌入式安装 + 密封胶 + 天地盖压紧 → 三层防水

3.1 12V Low-Voltage Waterproof LED Module（12V 低压防水 LED 模组）

  类型: 背发光专用高亮 LED 模组（非灯带——模组光效更均匀）

  参数:
    电压: 12V DC（低压安全，全球通用）
    防水: IP65（标准） / IP67（浸水级，选配）
    LED芯片: SMD 2835
    单模组功率: 1.5-3W
    排布密度: 每 100mm² 一个模组（视字大小调整）
    色温: 2700K（暖黄）/ 3000K（暖白）/ 4000K（自然白）
           5000K（冷白）/ 6500K（日光白）/ RGB幻彩（选配）
    光通量: 100-150 lm/模组
    寿命: >50,000 小时（MeanWell 驱动下）

  排布规则:
    - 每个字的轮廓内侧一圈密集排布（确保轮廓光晕饱满）
    - 填充区域均匀分布（间距 80-120mm，消除暗区）
    - 转角处加密（转角是最容易出现暗区的位置）
    - 避开水晶底板的螺丝孔/出线孔位置


3.2 Cable Outlet（精密出线孔）

  定义: 在不锈钢天地盖或围边底部开一个孔，让电源线从内部穿出

  标准:
    孔径: 8-12mm（根据线束直径）
    位置: 字壳底部中间偏后位置（安装后不可见）
    防护: 出线孔配防水锁头（PG9/PG11 防水电缆接头）
    密封: 锁头内侧垫 EPDM 橡胶圈 → 拧紧后 IP65

  "A cable outlet that's not sealed is a water inlet pretending
   to be a cable outlet. We use PG-rated waterproof glands.
   Your sign will outlast three buildings."


3.3 不锈钢天地盖无螺丝工艺（Interlocking Tool-Free Backplate）

  定义: 字壳背部封板采用嵌入式卡扣设计，无需螺丝

  结构:
    - 上盖（Face Side）: 不锈钢面板，与围边激光焊接一体
    - 下盖（Back Side）: 不锈钢/PVC 封板，Interlocking 卡扣固定

  安装流程:
    ① 排好 LED 模组 → 接好线 → 穿过 Cable Outlet
    ② 水晶底板嵌入围边凹槽 → 打密封胶
    ③ 天地盖从底部滑入 → 卡扣自动锁紧
    ④ PG 防水锁头锁死出线孔

  维护流程（反向操作）:
    ① 用扁口螺丝刀轻撬卡扣释放
    ② 天地盖滑出 → 露出内部所有组件
    ③ 换 LED / 换电源 / 清理 → 10 秒
    ④ 卡扣盖回去 → 完成

  "Your competitor's sign: screws rust → drill out → drill stripped
   → sign destroyed. Our sign: 10 seconds with a flathead screwdriver.
   Still works. Still sealed. Still beautiful."

4.1 Pin Spacers（精密撑脚/悬浮支架）

  定义: 304 不锈钢圆管或六角柱，从字壳背面伸出，撑开 15-30mm 间隙

  参数:
    材质: 304 不锈钢（默认）/ 316# 不锈钢（沿海/室外选配）
    直径: 6-10mm（标准）/ 12mm（大字重载）
    长度: 15-30mm（标准），可定制 10-50mm
    固定: 一端焊接在字壳背面，一端攻内螺纹（M6/M8）
    墙面侧: 304 不锈钢膨胀螺栓 + 尼龙防松垫片

  安装流程:
    ① 墙面钻孔 → 塞入膨胀螺栓
    ② 将字壳的 Pin Spacer 对准螺栓 → 拧紧
    ③ 每个字 2-6 个 Pin（视字大小而定）
    ④ 调整水平 → 锁紧 → 完成

  悬浮间隙的选择:
    ┌─────────────┬──────────────┬──────────────────────────────┐
    │ 间隙           │ 光晕效果       │ 适用场景                      │
    ├─────────────┼──────────────┼──────────────────────────────┤
    │ 15mm         │ 紧凑型光晕     │ 室内走廊/空间受限处           │
    │ 20mm         │ 标准光晕       │ 通用推荐 ✅                    │
    │ 25-30mm      │ 宽大光晕       │ 大门头/需要强烈视觉冲击        │
    │ > 30mm       │ 超宽光晕       │ 特殊设计需求（额外加固）       │
    └─────────────┴──────────────┴──────────────────────────────┘

  英国订单典型值: 20-25mm（标准大气的日冕效果）


4.2 The Halo Effect（日冕光晕——背发光的终极美学）

  当所有组件就位:

    1. MeanWell 电源通电 → 12V DC 送入字壳内部
    2. LED 模组点亮 → 光线射向 10-15mm 水晶亚克力底板
    3. 光线在厚亚克力内部经过多次漫反射 → 均匀分散
    4. 光线从水晶底板背面出射 → 打在墙面上
    5. 墙面反射出一个比字的外轮廓大 5-15mm 的光晕轮廓
    6. 不锈钢字壳本身保持暗色（不发光）→ 光与字之间的对比
       → 形成"字在墙上悬浮，光从字背后渗出"的视觉

  "Halo-lit is not backlit. Backlit is a light bulb behind a sign.
   Halo-lit is a letter casting its own shadow made of light.
   The letter is dark. The wall around it glows.
   It's the closest thing to levitation that signage has ever achieved."
