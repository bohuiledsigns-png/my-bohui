1.1 系统架构

  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
  │ LED Driver   │───▶│  Outline     │───▶│  字壳边缘光   │
  │ (恒流源)     │    │  Controller  │    │  (不锈钢)     │
  └──────────────┘    └──────────────┘    └──────────────┘
        │
        ▼
  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
  │ PWM Dimmer   │───▶│  Fill        │───▶│  芯板内部光   │
  │ (呼吸曲线)   │    │  Controller  │    │  (亚克力)     │
  └──────────────┘    └──────────────┘    └──────────────┘

  Outline Controller: 控制字壳边缘的轮廓光
    - 独立PWM通道，频率>200Hz（目视无闪烁）
    - 可单独配置: 亮度(0-100%), 呼吸频率(0.5-2Hz), 相位偏移(0-360°)
    - 典型值: 亮度70%, 频率0.8Hz, 相位0°

  Fill Controller: 控制芯板的填充光
    - 独立PWM通道，带软启动(Soft-Start)功能
    - 可配置: 步进深度(0.1-0.5mm), 折射角(30-45°), 色温(2700-6500K)
    - 典型值: 步进0.3mm, 折射角37°, 色温5000K

1.2 轮廓/填充独立控制的商业价值

  传统灯箱: 灯一开就亮，灯一关就灭，整面均匀，没有层次。
  GLOWFORGE: 轮廓线和内部填充分开受控，可独立调节。

  这意味着:
  - 白天关掉填充光，只保留轮廓 → 金属字壳的精密感
  - 黄昏填充光渐起 → 光的迁就感
  - 夜晚两层光融合 → 立体发光呼吸
  - 凌晨可切换为"守夜模式": 仅轮廓光10%亮度微亮

  【贰 · 1500# 拉丝不锈钢工艺标准】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2.1 材料选择标准

  牌号: SUS304 (日标) / AISI 304 (美标) / 06Cr19Ni10 (国标)
  厚度: 1.2mm-2.0mm (招牌字壳推荐1.5mm)
  表面: 2B原板 → 单向拉丝处理
  拉丝砂带: 1500# 氧化铝砂带 (日本NCA/韩国DEERFOS)
  研磨方向: 单向直线，与字壳最长边平行，禁止交叉研磨
  研磨速度: 12-15m/min (过快→发热变色，过慢→纹路过深)
  最终Ra: ≤0.4μm (触针式粗糙度仪检测)

2.2 激光切割参数 (GLOWFORGE推荐值)

  功率: 80-90% (500W光纤激光器)
  速度: 12-15mm/s (1.5mm厚)
  辅助气: 氮气 0.8MPa (防止氧化变色)
  焦点: 0 (材料表面)
  切口宽度: 0.12-0.15mm
  热影响区(HAZ): <0.05mm

  对比 — 竞争对手的CNC铣削:
  铣刀直径: 3.175mm (1/8")
  切口宽度: 3.3mm
  HAZ: 0.1-0.3mm
  需二次打磨: 是 (每字壳额外8-15分钟)
  砂带耗材成本: $4-8/字壳

2.3 R3-R5 圆角工艺

  加工方式: 激光切割一次性成型，转角自动圆弧插补
  圆角类型: G2连续 (曲率连续，无拐点)
  检测工具: R规(半径规) + 投影仪放大20×
  合格标准: 圆弧与直线段的切点肉眼不可辨，手摸无顿挫感

  "R3不是半径，是傲慢" 的工程含义:
  R3mm 是激光切割机在不降速的前提下能加工的最小稳定圆角
  <3mm → 必须降速50% → 生产效率砍半 → 成本翻倍
  因此R3代表着一个平衡点: 精密 × 高效 × 可复制

2.4 激光无缝焊接

  焊接类型: 激光搭接焊 (Laser Lap Welding)
  对接缝间隙: <0.05mm (CNC折弯后的装配精度)
  焊接速度: 30-40mm/s
  焊缝宽度: 0.3-0.5mm
  熔深: 0.3-0.5mm (不完全穿透，保持背面原色)
  后处理: 无需打磨，直接进入表面清洁

  优势对比:

  │ 是否需要   │ 是         │ 是         │ 否           │

  【叁 · 亚克力导光芯板工程标准】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3.1 材料

  基材: 浇铸亚克力板 (Cast PMMA), 非挤出板(Extruded)
  厚度: 12mm (标准), 8mm/15mm (可选)
  透光率: ≥92% (550nm 绿光波段)
  耐候: UV稳定剂添加, 3000h QUV测试无黄变
  供应商推荐: 三菱丽阳(日本) / 璐彩特(韩国) / 申威达(中国)

3.2 步进雕刻参数

  步进深度: 0.3mm/step
  步进数量: 32 steps (标准12mm板)
  总雕刻深度: 9.6mm (预留2.4mm基底)
  雕刻模式: 逐层递减 (圆台状截面)
  雕刻角度: 每层阶梯的反射面倾角 37°
  表面处理: 雕刻面火焰抛光 (恢复透光率到95%+)

3.3 光路原理

  入射 → [边缘入光]
     ↓
  第一层折射 → 光偏转 37° → 部分出射 (15%总光量)
     ↓
  第二层折射 → 光偏转 74° → 部分出射 (12%)
     ↓
  第三次折射... → 累计偏转 111° → 部分出射
     ↓ (共32层)
  最终 → 剩余 <5%光量 → 在芯板末端被吸光涂层吸收

  效果: 光在32次出射中均匀分布，无热点无暗区
  均匀度: ±8% 以内 (行业标准 ±20%)

3.4 防UV黄变方案

  "亚克力黄变" 是行业通病，博汇的解决方案:
  1. 使用浇铸级PMMA (非挤出级)—分子链更长，抗UV更强
  2. UV稳定剂按2.5%比例熔入基材 (行业标准1.5%)
  3. 光路设计避免UV波段 (400nm以下)在材料内反复折射
  4. LED选用低UV输出型 (4000-6500K光谱峰值偏移可见光波段)
