【第二级国防 · API 接口一键无缝热切换】
  三条命 · 死一条自动换一条

  三条主力 API 配置:
    主力战将: Gemini 2.5 Pro (getgoapi.com)  — 多模态看图+推理
    备胎一号: DeepSeek V3/R1 (deepseek.com)  — 纯文字最强文案
    备胎二号: Claude Sonnet 4 (getgoapi.com) — 多模态备用眼睛

  手动热切换步骤（当中转站暴雷时）:

  场景 A：主力 Gemini 挂掉 → 切 DeepSeek（纯文字模式）
    1. 打开 Hermes 后台 → 修改 provider/model 配置
    2. 把 model 从 gemini-2.5-pro 改为 deepseek-chat
    3. 把 endpoint 改为官方 API 或备用的中转地址
    4. 保存 → 刷新对话
    5. 粘贴第一级里的"提示词注入规则"
    6. 继续用文字出方案、写话术、报价
    注意：DeepSeek 看不到图——此时你在聊天框里用文字描述图片内容。

  场景 B：DeepSeek 也挂了 → 切 Claude Sonnet 4
    1. 同样步骤，把 model 改为 claude-sonnet-4-6
    2. Claude 支持多模态（看图）
    3. 同样注入 D 盘 Wiki 提示词
    4. 继续工作

  快速切换备忘:
    ① 深呼吸
    ② 改 model 名字
    ③ 改 endpoint
    ④ 粘贴提示词注入
    ⑤ 继续干活
    整个过程不超过 3 分钟。

【第三级国防 · 本地开源模型低成本部署】
  大不了回老家 · 不联网也能打

  适用场景:
    海外全面封锁所有 API 中转站
    国内的模型也限流/无法访问
    你需要在断网环境下继续写话术、看图纸

  推荐本地模型:
    Qwen2.5-VL-7B   (~5GB)  看图+读图纸  (Ollama一键部署)
    Qwen2.5-VL-72B  (~45GB) 强视觉       (需GPU)
    Llama-3.2-Vision(~8GB)  看图          (Ollama一键)
    DeepSeek-R1     (~7GB)  纯文字        (低门槛)

  部署步骤（以 Qwen2.5-VL-7B 为例）:
    Step 1: 安装 Ollama
      打开 https://ollama.com/download → 下载 Windows 版 → 一键安装
    Step 2: 拉取模型
      终端输入: ollama pull qwen2.5-vl:7b
    Step 3: 启动模型
      ollama run qwen2.5-vl:7b
    Step 4: 喂 D 盘 Wiki
      复制第一级里的"提示词注入规则"贴进去
    Step 5: 传图分析
      直接把图纸拖进终端窗口 → 模型自动识别
    Step 6: 继续工作
      不需要联网。不需要 API Key。不需要中转站。

  硬件要求:
    CPU: 现代 i5/i7 或 Ryzen 5/7
    内存: 16GB+（推荐 32GB）
    GPU: 有 NVIDIA 显卡更好，没有也能跑
    硬盘: 剩余 10GB+

  终极离线工作流:
    就算没有 AI 模型，你的 D 盘里有 7 层 Wiki + 10 个话术模板。
    手动打开这些文件，复制粘贴话术、改客户名、改参数，直接发 WhatsApp。
    模型只是加速器，不是必需品。知识在手，断网也能打。
