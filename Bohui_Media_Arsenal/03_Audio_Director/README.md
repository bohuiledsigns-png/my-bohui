# 03_Audio_Director — 剪辑节奏与旁白脚本指导

本库存放博汇视频的音频节奏标准。

## 已收录音频方案 (见主弹药库文件)
- 15秒版旁白: "Light with a passport"
- 9秒版旁白: "1500# stainless. R3 corners. 37° refraction."
- 4种BGM类型: Lo-Fi / 管弦打击 / ASMR / 幽默人声

## 音频标准
- BGM音量: -18dB (不压过VO)
- VO音量: -6dB (清晰但不轰)
- ASMR效果: 近场麦克风, 距离 <5cm
- 节奏: 每3秒一个场景切换
- 收尾: fade out 0.5s + 品牌音效标识

"signage background music" / "premium cinematic logo" /
"corporate lofi beat" / "slow cinematic tension"
