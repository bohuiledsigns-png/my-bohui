# 🧠 记忆进化系统

## 架构

```
memory/
├── README.md              ← 这个文件，系统说明
├── MEMORY.md              ← 记忆索引（自动加载）
│
├── user/                  ← 关于你
│   └── profile.md         ← 个人信息/偏好/目标
│
├── system/                ← 关于我的配置和进化
│   ├── evolution-log.md   ← 进化日志（能力更新）
│   └── commands.md        ← 你教我的指令/工作流
│
├── projects/              ← 每个项目一个文件
│   ├── _index.md          ← 项目总目录
│   └── <project-name>.md  ← 项目详情
│
└── knowledge/             ← 知识库（积累的专业知识）
    ├── _index.md
    └── <topic>.md
```

## 如何进化

1. **每次会话自动加载记忆**
2. **你说"学习：XXX"** → 我记入知识库
3. **你说"记住：XXX"** → 我记入用户偏好
4. **项目进展自动更新**
5. **老记忆自动归档但保留**

## 超长记忆机制

- `MEMORY.md` 是索引，每次会话都加载
- 详细内容存在子文件，需要时我再去读
- 这样既记得住，又不会撑爆上下文
